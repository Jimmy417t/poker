# 德州撲克遊戲系統

## 一、動機與目標
實作命令列版德州撲克，包含牌庫系統、牌型判斷、下注流程與 AI 對手，練習 C++ 物件導向與演算法設計。

## 二、預期功能
- 52 張牌庫（洗牌、發牌）
- 人類玩家 vs AI 玩家
- 棄牌 / 跟注 / 加注
- 10 種牌型判斷
- 籌碼管理

## 三、使用技術
C++17、vector、類別、繼承、隨機洗牌

## 四、時程規劃
| 週次 | 內容 | 產出 |
|------|------|------|
| 1 | Card、Deck、Player | 發牌系統 |
| 2 | 牌型判斷 | 正確判斷牌型 |
| 3 | AI 與下注 | AI 自動決策 |
| 4 | 整合測試 | 完整遊戲 |

## 五、課程關聯
類別、繼承、列舉、容器、演算法、隨機數
